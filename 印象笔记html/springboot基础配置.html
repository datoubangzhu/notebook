@Configuration标注在类上，相当于把该类作为spring的xml配置文件中的<beans>，作用为：配置spring容器(应用上下文)
package com.dxz.demo.configuration;
import org.springframework.context.annotation.Configuration;
@Configuration
public class TestConfiguration {
    public TestConfiguration() {
        System.out.println("TestConfiguration容器启动初始化。。。");
    }
}

三.继承自ApplicationContextAware接口
继承ApplicationContextAware接口的类会在实例化时自动执行接口中setApplicationContext(ApplicationContext    applicationContext)方法，通过此方法可以对spring上下文进行管理，例如ApplicationContext.getBeanDefinationName()用于展示容器中所有bean的名字，可以通过非注解的方式执行更多有关类的操作
