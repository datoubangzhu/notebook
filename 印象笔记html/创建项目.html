创建项目

创建springboot项目（自带maven）：
File->New Project->Spring Initializr ->web->web



创建spring项目  （自带maven）
File->New Project->maven->Create from archetype -> org.apache.maven.archetypes:maven-archetype-webapp
(1)spring依赖：  
<dependency>
 <groupId>org.springframework</groupId>
 <artifactId>spring-context</artifactId>
 <version>3.2.17.RELEASE</version>
</dependency>
(2)springmvc依赖：
<dependency>
    <groupId>org.springframework</groupId>
    <artifactId>spring-webmvc</artifactId>
    <version>3.2.17.RELEASE</version>
</dependency>




创建带有maven的web servlet项目:
     (1)先新建maven web servlet项目
         File->New Project->maven->org.apache.cocoon:cocoon-22-archetype-webapp ;   
         右击项目->Add Framework Support ->Web Application   //生成web目录，和eclipse项目中的的web目录类似（含index.jsp）
     (2)添加web.xml
         File->Project Structure->Facets，在中间web中找到自己刚刚创建的servlet项目，在右侧Deployment Descriptors中，点击"+"，会提示web.xml
     (3)创建Servlet要进行配置：
<dependency>
<groupId>javax.servlet</groupId>
<artifactId>javax.servlet-api</artifactId>
<version>3.1.0</version>
</dependency>
